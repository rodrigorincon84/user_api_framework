## Stack
* [poetry](https://python-poetry.org/) - Python project manager
* [pytest](https://pytest.org/) - Testing framework
* [pytest-xdist](https://pypi.org/project/pytest-xdist/) - Parallel execution
* [httpx](https://www.python-httpx.org/) - Http client
* [httpx-retries](https://will-ockmore.github.io/httpx-retries/) - Re-execute http requests
* [pydantic](https://pydantic.dev/docs/validation/latest/get-started/) - Data conversion 
* [assertpy](https://github.com/assertpy/assertpy) - Assertions
